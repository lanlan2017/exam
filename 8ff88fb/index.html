考点3

关于匿名内部类叙述正确的是？ ( )

  • A 匿名内部类可以继承一个基类,不可以实现一个接口
  • B 匿名内部类不可以定义构造器
  • C 匿名内部类不能用于形参
  • D 以上说法都不正确

解析

显示答案/隐藏答案正确答案: B

匿名内部类不能有构造器

由于构造器的名字必须与类名相同，而匿名类没有类名，所以匿名类不能有构造器。
