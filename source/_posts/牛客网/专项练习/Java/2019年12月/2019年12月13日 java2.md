# 题目2 考点:求余运算
以下代码段执行后的输出结果为
```java
public class Test {
    public static void main(String args[]) {
        int x = -5;
        int y = -12;
        System.out.println(y % x);
    }
}
```
- A -1
- B 2
- C 1
- D -2

## 解析
<details><summary>显示答案/隐藏答案</summary>正确答案: D</details>

Y(被除数) % X(除数) 当除数与被除数的符号相同时，取余和取模的结果是完全相同的； 当除数与被除数的符号不相同时，结果不同。具体说，**取余结果的符号与%前面的数的符号(被除数)相同**；
